# Overloading

## 개요
오버로딩은 객체 지향 프로그래밍에서 다형성에 의하여 같은 이름을 가진 메소드를 연산자를 다르게 하여 여러번 재정의해서 사용하는 것을 의미한다.

## 특징
1. 메소드 이름이 같아야 한다.
2. 리턴형이 같아도 되고 달라도 된다.
3. 파라미터 갯수가 달라야한다.
4. 만약 파라미터 갯수가 같은 경우, 데이터 타입이 달라야 한다.
