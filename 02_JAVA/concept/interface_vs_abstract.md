# 추상클래스 vs 인터페이스 차이

실제로 면접시에 cs질문 가장 기초로써 많이 나오는 질문이다.
쉽게 말하자면,
- 추상클래스는 단순히 반복되는 내용을 공통화시켜 중복된 코드를 줄이는 것에 목적
- 인터페이스는 유지보수 및 관리 목적에서 코드를 공통화시키는 것에 목적

정도로 이해하면 될 것 같다.
